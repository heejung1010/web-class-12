자바스크립트의 객체 시스템

자바스크립트는 브라우저 자체를 객체로 이해하고 있으며 최상위 객체를 window객체라고 부른다

그리고 그 최상위 객체 window아래에는 여러 객체들이 포함되어 있다

  • document -> 정보들을 표현하는 문서객체 (문서 관리 비서실)
  • location -> 웹페이지의 위치 정보를 표현하는 객체 (내비게이션)
  • history -> 이동했던 페이지들에 대한 정보를 관리하는 객체
  • console -> 개발자 툴에 있는 콘솔 객체 (개발자에게만 들리는 소리)
  • navigator -> 브라우저의 정보를 담고있는 객체(사용자가 크롬, 폰, 사파리... 브라우저 프로필 정보 궁금)
